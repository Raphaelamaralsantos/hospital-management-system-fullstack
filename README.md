# Hospital Website - Rafael & Davi

## 🚀 Visão Geral
Site moderno e responsivo para hospital com páginas separadas:
- **Home** (`index.html`): Landing page
- **Serviços** (`servicos.html`): 4 serviços detalhados 
- **Login** (`login.html`): Área do paciente
- **Agendar** (`agendar.html`): Formulário de consulta

## ✨ Funcionalidades
- ✅ Design glassmorphism/gradientes responsivo (mobile-first)
- ✅ Navegação fixa consistente
- ✅ Login demo: `paciente@test.com` / `123456`
- ✅ Agendamento requer login (localStorage)
- ✅ Animações CSS + hover effects
- ✅ Formulários funcionais JS

## 📁 Estrutura
```
projeto.web.rafael-davi/
├── index.html         # Home
├── servicos.html      # Serviços (4 cards)
├── login.html         # Login área paciente
├── agendar.html       # Agendar consulta
├── styles.css         # Estilos compartilhados
├── Tela-home.html     # Versão original (backup)
└── img/               # Imagens (fundo.webp, htp.webp, hospital.webp)
```

## 🎮 Como Usar
1. Abra `index.html` no navegador
2. Navegue pelos links do menu
3. **Login demo**: paciente@test.com / 123456
4. Agende consulta em \"Agendar\" (após login)

## 🔧 Tecnologias
- HTML5 semântico
- CSS3 (Grid, Flexbox, Animations, Backdrop-filter)
- Vanilla JavaScript (localStorage, forms)
- Google Fonts (Poppins)
- Totalmente responsivo

## 📱 Responsivo
- Desktop, Tablet, Mobile
- Clamp() para tipografia fluida
- Media queries otimizadas
